# RPG Box プロジェクト概要

## 1. プロジェクトについて

**RPG Box**は、プログラミング経験がないユーザーでも直感的に本格的な2D RPGを作成・公開できるブラウザベースのゲームエディタです。

### 主な特徴

- **直感的なエディタ**: マップ、データ、UIをGUIで操作可能
- **ブラウザ完結**: インストール不要、どこでも開発可能
- **高い拡張性**: JavaScriptによるスクリプト制御が可能

## 2. 技術スタック

本プロジェクトは以下のモダンなWeb技術を用いて開発されています。

### フロントエンド

- **コアフレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS, Shadcn/UI
- **状態管理**: Zustand + Immer
- **エディタコンポーネント**: Monaco Editor (コード編集), dnd-kit (ドラッグ&ドロップ)

### データ保存

- **ローカル**: IndexedDB (idb) - プロジェクトデータの保存
- **一時保存**: LocalStorage - クラッシュ復旧用

## 3. ディレクトリ構造


## 4. 主要機能モジュール

### データエディター (Data Editor)

キャラクター、アイテム、スキルなどのゲームデータを管理します。

- **柔軟なフィールド設計**: 数値、文字列、画像など多彩な型に対応
- **継承クラスパターン**: 拡張性の高いデータ構造

### マップエディター (Map Editor)

タイルセットを使用したマップ制作機能を提供します。

- **レイヤー機能**: タイル、オブジェクト、イベントの階層管理
- **ツール**: ペン、塗りつぶし、スタンプなどの描画ツール

### スクリプトエディター (Script Editor)

ゲームロジックをJavaScriptで記述できます。

- **Monaco Editor採用**: 高機能なコード編集
- **専用API**: ゲーム内変数やオブジェクト操作用のAPIを提供

### ゲームエンジン (Game Engine)

作成したゲームをブラウザ上で実行するためのランタイムです。

- **Canvas/WebGL**: 高速な描画処理
- **コンポーネントシステム**: オブジェクトの振る舞いを部品化
