# Test Play v1 実装計画

> **エージェント向け:** superpowers:subagent-driven-development（サブエージェント利用可能時）または superpowers:executing-plans で実行すること。各ステップはチェックボックス (`- [ ]`) で進捗管理。

**ゴール:** ▶ ボタンでテストプレイを起動し、マップ上をプレイヤーが歩ける・トリガーが発火する・スクリプトが実行される状態にする。

**方針:** インフラは実装済み（TestPlayOverlay, buildProjectData, GameRuntime, GameWorld）。このプランでは動作を妨げている6つの不整合を修正する: startPosition廃止、移動完了通知のワイヤリング、GameContextのイベント間永続化、キャンバスリサイズ修正、プレイヤー不在警告。

**技術スタック:** TypeScript, WebGL (twgl.js), Zustand, React, Jest

**仕様書:** `docs/superpowers/specs/2026-03-14-test-play-v1-design.md`

```typescript
update(dt: number, input: InputManager): { obj: RuntimeObject; fromX: number; fromY: number }[] {
  const completions: { obj: RuntimeObject; fromX: number; fromY: number }[] = [];

  for (const obj of this.objects) {
    if (obj.isMoving) {
      const fromX = obj.gridX;
      const fromY = obj.gridY;
      this.advanceMovement(obj, dt);
      if (!obj.isMoving) {
        completions.push({ obj, fromX, fromY });
      }
      continue;
    }

    const dir = this.resolveDirection(obj, input);
    if (!dir) continue;

    obj.facing = dir;
    const delta = DIR_DELTA[dir];
    const toX = obj.gridX + delta.dx;
    const toY = obj.gridY + delta.dy;

    if (this.canMove(obj.gridX, obj.gridY, toX, toY)) {
      this.startMove(obj, toX, toY);
    }
  }
  return completions;
}
```

```typescript

    return {
      scripts: this.projectData.scripts,
      variables: this.projectData.variables.map((v) => ({
        id: v.id,
        name: v.name,
        type: v.fieldType.type,
        defaultValue: v.initialValue,
      })),
      classes: this.projectData.classes?.map((c) => ({
        id: c.id,
        name: c.name,
        fields: c.fields?.map((f) => ({ id: f.id, fieldType: f.type })) ?? [],
      })) ?? [],
      dataTypes: this.projectData.dataTypes?.map((dt) => ({ id: dt.id, name: dt.name })) ?? [],
      dataEntries: this.projectData.dataEntries ?? {},
    };
  }
```
